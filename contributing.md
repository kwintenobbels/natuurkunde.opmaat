## Bijdragen aan Natuurkunde Op Maat

Iedereen kan bijdragen aan het leermateriaal. 

### Werkwijze: issues en branches

We werken via **korte branches** die vertrekken en eindigen in de main-branch. 
De eenvoudigste werkwijze is middels 'create a branch for this issue': 

- **Maak een issue**: Start elke aanpassing met een issue, hierin kan je kort beschrijven wat je wijzigt. Bijv: dtfout-horizontaleworp 
- **Branch per issue**: Gebruik "Create branch from issue" rechts onderaan, Github maakt dat een branch voor het issue en nummert deze automatisch. Momenteel zitten we op 141-dtfout-horizontaleworp 
- **Voor wijzigingen en verbetering door**: In de branch kan je de code aanpassen. 
- **Zet een Pull request open**:
  - druk op de knop 'create pull request' 
  - Beschrijf kort wat er verandert en waarom.
  - Je wijzigingen worden ingevoegd en verschijnen online.
